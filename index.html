ActiveImage

jQuery Plugin

Fork me on GitHub

ActiveImage is a jQuery plugin that will allow any inline image
to be visually popped out of the content and expanded to fill the screen.

Usage
Just add the 'active-image.js' file to your project:
<script src="active-image.js"></script>
And call the plugin:
$( 'img' ).activeImage();
Thats it! The plugin handles the math so that your image retains its aspec ratio and fits within the viewport.
